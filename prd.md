# KidsEdu - 유아 교육 앱 PRD v2.0

## 1. 개요

4~5세 유아를 위한 숫자(1-10), 한글(자음/모음), 영문(A-Z) 학습 및 다양한 미니 게임을 포함한 **태블릿 최적화 PWA** 서비스.

**v2.0 핵심 개선 방향:**
- 아이들에게 더 친숙하고 몰입감 있는 비주얼 디자인
- 따라쓰기를 실제 텍스트 위에 직접 오버레이하여 쓸 수 있는 UX
- 게임 콘텐츠 대폭 확충 (3종 → 8종)
- LetterSchool, Writing Wizard, Khan Academy Kids, 핑크퐁 수준의 완성도 벤치마크

## 2. 벤치마크 분석

### 2.1 참고 앱

| 앱 | 핵심 강점 | 적용 포인트 |
|---|---|---|
| **LetterSchool** | 3단계 반복 트레이싱 (따라가기 → 점 잇기 → 자유쓰기), 글자 위 직접 오버레이 | 단계별 트레이싱 시스템 도입 |
| **Writing Wizard** | 50+ 애니메이션 스티커, 5-star 난이도 시스템, 부모 리포트 | 보상 스티커, 진행도 리포트 |

| **핑크퐁** | 동요 연계 학습, 반복 노출, 밝은 색감 | 사운드 기반 학습 강화 |

### 2.2 디자인 벤치마크 원칙 (UX for Kids)
- **큰 터치 타겟**: 모든 인터랙티브 요소 최소 56px (업계 권장 44px 이상)
- **단순한 레이아웃**: 화면당 선택지 최대 4~6개
- **즉각적 피드백**: 모든 터치에 시각 + 청각 반응
- **짧은 세션**: 한 학습 단위 2~3분 이내
- **긍정적 강화**: 실패 없는 설계 — 틀려도 격려, 재시도 유도
- **부모 게이트**: 설정/외부 링크는 부모 잠금 (간단한 수학 문제 등)

## 3. 타겟 사용자

### 3.1 직접 사용자: 4~5세 유아
- 한글 자음/모음을 처음 접하는 단계
- 숫자 1~10 인지 및 쓰기 학습 단계
- 영어 알파벳 첫 노출 단계
- 터치 기반 인터랙션에 익숙하나 복잡한 제스처 어려움
- 짧은 집중력 (3~5분), 즉각적 보상에 민감

### 3.2 간접 사용자: 부모/보호자
- 자녀 학습 진행도 확인 욕구
- 광고 없는 안전한 환경 요구
- 화면 시간 관리 필요

## 4. 핵심 기능

### 4.1 숫자 학습 (1-10) — P0

#### 4.1.1 숫자 목록 화면
- 1~10 숫자를 2열 그리드로 표시
- 각 카드: 큰 숫자 + 해당 개수만큼의 귀여운 오브젝트 일러스트(SVG)
- 완료된 숫자는 금색 별 뱃지 표시
- 카드 터치 시 바운스 애니메이션 + 해당 숫자 TTS 재생

#### 4.1.2 숫자 학습 상세 화면
- **상단**: 큰 숫자 표시(120px+) + TTS 자동 재생 (숫자 읽어주기)
- **중앙**: 오버레이 따라쓰기 캔버스 (아래 4.4절 상세)
- **하단**: 카운팅 인터랙션 — 오브젝트를 하나씩 터치하면 숫자가 올라가며 카운팅
- 따라쓰기 3단계 완료 시 → 별 보상 + 축하 애니메이션 + 다음 숫자 이동
- 이전/다음 숫자 네비게이션 (스와이프 또는 화살표)

#### 4.1.3 숫자 데이터
```
1 = 하나/One, 오브젝트: 사과
2 = 둘/Two, 오브젝트: 체리
3 = 셋/Three, 오브젝트: 별
...10 = 열/Ten, 오브젝트: 하트
```

### 4.2 한글 학습 (자음 ㄱ-ㅎ, 모음 ㅏ-ㅣ) — P0

#### 4.2.1 한글 목록 화면
- 자음(ㄱ-ㅎ 14개) / 모음(ㅏ-ㅣ 10개) 탭 전환
- 3열 카드 그리드: 글자 + 대표 단어 + 연관 이미지(SVG)
- 완료 항목 별 뱃지 표시
- 카드 터치 시 글자 발음 TTS 재생

#### 4.2.2 한글 학습 상세 화면
- **상단**: 큰 글자(100px+) + 대표 단어 + 연관 이미지
- **중앙**: 오버레이 따라쓰기 캔버스 — 획순 가이드 포함
- **하단**: 대표 단어 카드 (단어 + 이미지 + TTS)
  - 예: ㄱ → "기린" + 기린 SVG + "기린" TTS
- 획순 애니메이션: 첫 단계에서 손가락 아이콘이 획순대로 이동하며 안내

#### 4.2.3 한글 데이터
```
ㄱ = 기역, 대표단어: 기린, 획순: [좌상→우상→하단]
ㄴ = 니은, 대표단어: 나비, 획순: [좌하→우하→우상]
ㄷ = 디귿, 대표단어: 다람쥐, 획순: [...]
...
ㅏ = 아, 대표단어: 아이스크림
ㅓ = 어, 대표단어: 어린이
...
```

### 4.3 영문 학습 (A-Z) — P0

#### 4.3.1 영문 목록 화면
- A-Z 알파벳 4열 카드 그리드
- 각 카드: 대문자 + 소문자 + 대표 단어 이미지
- 완료 항목 별 뱃지 표시
- 카드 터치 시 알파벳 발음 TTS 재생 (en-US)

#### 4.3.2 영문 학습 상세 화면
- **상단**: 대문자/소문자 표시(100px+) + 대표 단어 + 이미지
- **중앙**: 오버레이 따라쓰기 캔버스 (대문자 → 소문자 순서)
- **하단**: 대표 단어 카드 + TTS (영어 발음 + 한글 뜻)
  - 예: A/a → "Apple" + 사과 SVG + "애플" TTS

#### 4.3.3 영문 데이터
```
A/a = Apple(사과), B/b = Bear(곰), C/c = Cat(고양이)...
```

### 4.4 오버레이 따라쓰기 시스템 — P0 (핵심 개선)

> **벤치마크**: LetterSchool의 3단계 트레이싱 + Writing Wizard의 오버레이 방식

기존 "캔버스 옆에 가이드" 방식에서 **실제 글자 위에 직접 따라 쓰는 오버레이 방식**으로 전환.

#### 4.4.1 3단계 트레이싱 시스템

**1단계 — 따라가기 (Guided Trace)**
- 글자가 연한 색(opacity 0.3)으로 표시
- 획순대로 점선 경로가 표시됨
- 시작점에 빛나는 원형 인디케이터가 표시됨
- 사용자가 인디케이터를 터치하면 손가락을 따라 획이 그려짐
- 경로를 벗어나면 부드럽게 다시 안내 (벌칙 없음, "다시 해볼까?" 격려)
- 각 획 완료 시 짧은 효과음 + 반짝임 이펙트

**2단계 — 점 잇기 (Dot-to-Dot)**
- 글자가 연한 색으로 표시
- 획순 경로 위에 번호 매긴 점들이 표시됨 (각 획 시작점/꺾이는점/끝점)
- 사용자가 점 순서대로 연결하며 따라 씀
- 점에 도달하면 "팡" 효과 + 다음 점 활성화

**3단계 — 자유 쓰기 (Free Write)**
- 글자가 매우 연하게(opacity 0.1) 배경에 표시
- 가이드 없이 자유롭게 쓰기
- 완성도 판정: 글자 영역 60% 이상 커버 시 성공
- 성공 시 글자가 무지개색으로 변하며 축하 애니메이션

#### 4.4.2 캔버스 기술 스펙
- HTML5 Canvas, 2개 레이어: 배경 글자 레이어(가이드) + 사용자 입력 레이어
- DPI 스케일링 (window.devicePixelRatio)
- 터치 이벤트: PointerEvent 기반 (터치/마우스/펜 통합)
- 선 굵기: 12px (아동 손가락 두께 고려), 둥근 끝(round cap)
- 부드러운 곡선: quadraticCurveTo
- touch-action: none (스크롤 방지)
- 지우기/다시쓰기/다음 단계 버튼
- 캔버스 크기: 반응형, 최소 280px × 280px, 최대 400px × 400px

#### 4.4.3 따라쓰기 피드백
- **진행 중**: 펜 경로에 반짝이는 파티클 트레일
- **획 완료**: 짧은 "딩" 효과음 + 반짝임
- **단계 완료**: 마스코트 격려 ("잘했어!") + 별 1개
- **전체 완료** (3단계 모두): 축하 컨페티 + 별 3개 + 레벨업 체크

### 4.5 미니 게임 — P0 (대폭 확충)

> **벤치마크**: Khan Academy Kids, Sago Mini World, 핑크퐁의 다양한 게임 유형 참고

기존 3종에서 **8종**으로 확충. 학습 내용과 연계된 게임 설계.

#### 4.5.1 게임 목록 화면
- 2열 카드 그리드, 각 게임 카드에 큰 일러스트 + 게임 이름 + 난이도 별표시
- 잠금된 게임: 특정 학습 진행도 달성 시 해금 (동기 부여)
- 카드 터치 시 게임 미리보기 애니메이션

#### 4.5.2 게임 상세

**게임 1: 글자-이미지 매칭 (Memory Match)**
- 카드 뒤집기 메모리 게임
- 4쌍(쉬움) / 6쌍(보통) / 8쌍(어려움) 난이도
- 글자 카드 ↔ 해당 이미지 카드 매칭
- 매칭 성공 시 카드가 사라지며 효과음
- 전체 클리어 시 소요 시간 표시 + 별 보상
- 카테고리 선택: 숫자/한글/영어

**게임 2: 소리 퀴즈 (Sound Quiz)**
- TTS로 글자/숫자/단어 읽어주기
- 3개 보기 중 맞는 것 선택
- 맞으면: 마스코트 축하 + 별 + 정답 글자 반짝임
- 틀리면: 마스코트 격려 ("다시 들어볼까?") + 재생 버튼 + 재시도 (무제한)
- 10문제 1세트, 카테고리 선택 가능

**게임 3: 순서 맞추기 (Sorting)**
- 3~5개 숫자/글자를 올바른 순서로 정렬
- 드래그앤드롭 (PointerEvent 기반, 라이브러리 미사용)
- 올바른 위치에 놓으면 "착!" 효과 + 초록 하이라이트
- 잘못된 위치는 부드럽게 원래 자리로 복귀
- 숫자(1→10), 자음(ㄱ→ㅎ), 알파벳(A→Z) 순서

**게임 4: 풍선 터뜨리기 (Balloon Pop)** ⭐ 신규
- 화면에 풍선이 떠오름 (각 풍선에 글자/숫자 표시)
- TTS로 "3을 찾아보세요!" 안내
- 맞는 풍선 터치 → 풍선 터지는 애니메이션 + 컨페티
- 틀린 풍선 터치 → 풍선이 흔들리며 "아니야~" 효과음
- 제한시간 없음, 연속 10문제

**게임 5: 색칠하기 (Coloring)** ⭐ 신규
- 글자/숫자 윤곽선이 표시됨
- 하단에 6색 팔레트 (빨/주/노/초/파/보)
- 색상 선택 후 영역 터치하면 색칠됨
- 자유 색칠 모드: 빈 캔버스 + 다양한 도구 (붓/크레용/스프레이)
- 완성 작품은 갤러리에 저장 (LocalStorage)

**게임 6: 버블 매칭 (Bubble Match)** ⭐ 신규
- 화면에 글자/숫자 버블이 무작위 배치
- 같은 글자 버블 2개를 연속 터치하면 버블이 합쳐지며 사라짐
- 모든 버블 쌍 제거 시 클리어
- 시간 제한 없음, 힌트 버튼 제공 (같은 쌍 하이라이트)

**게임 7: 그림자 맞추기 (Shadow Match)** ⭐ 신규
- 상단에 컬러 오브젝트 이미지
- 하단에 3~4개 그림자(실루엣) 보기
- 올바른 그림자에 오브젝트를 드래그앤드롭
- 학습 연계: 오브젝트의 첫 글자 표시 (예: 사과 → ㅅ)

**게임 8: 단어 만들기 (Word Builder)** ⭐ 신규
- 이미지가 표시됨 (예: 나비)
- 하단에 글자 블록들이 흩어져 있음 (ㄴ, ㅏ, ㅂ, ㅣ + 오답 글자)
- 올바른 순서로 글자 블록을 빈칸에 드래그
- 완성 시 단어 TTS 재생 + 축하 애니메이션
- 한글 자음+모음 조합 학습 (심화 콘텐츠)

#### 4.5.3 게임 공통 사양
- 모든 게임 상태 머신: ready → playing → success/fail → reward
- 게임 결과: 별 1~3개 (정확도 기반)
- 게임 해금 조건: 해당 카테고리 학습 30% 이상 진행
- 재시도 무제한, 실패 패널티 없음
- 사운드 ON/OFF 토글
- 게임별 최고 기록 저장

### 4.6 게이미피케이션 시스템 — P0

#### 4.6.1 보상 체계
| 행동 | 보상 |
|---|---|
| 따라쓰기 1단계 완료 | 별 1개 |
| 따라쓰기 3단계 전체 완료 | 별 3개 + 스티커 |
| 미니 게임 클리어 | 별 1~3개 (정확도) |
| 카테고리 100% 완료 | 특별 스티커 + 트로피 |
| 3일 연속 학습 | 보너스 별 5개 |
| 7일 연속 학습 | 특별 캐릭터 의상 해금 |

#### 4.6.2 레벨 시스템
- 별 10개 → 레벨 1, 별 30개 → 레벨 2, 별 60개 → 레벨 3...
- 레벨업 시 화려한 축하 애니메이션 + 새로운 캐릭터 표정 해금
- 최대 레벨 10 (모든 콘텐츠 완료 기준)

#### 4.6.3 스티커 컬렉션
- 학습 진행에 따라 스티커 획득
- 스티커북 화면에서 수집한 스티커 확인
- 카테고리별 스티커 테마: 숫자(동물), 한글(음식), 영어(탈것)

#### 4.6.4 캐릭터 성장
- 마스코트 캐릭터 (귀여운 곰 "또리")
- 레벨에 따라 외형 변화 (모자, 배경 색상 등)
- 학습 중 캐릭터가 동행하며 격려 멘트

### 4.7 홈 화면 — P0

#### 4.7.1 레이아웃
- **상단 영역**: 마스코트 캐릭터 + 시간대별 인사 ("좋은 아침!" 등) + 레벨 뱃지 + 별 개수
- **메인 영역**: 학습 카테고리 카드 4개 (숫자/한글/영어/게임)
  - 각 카드: 카테고리 SVG 아이콘 + 이름 + 원형 진행도 링
  - Framer Motion stagger 입장 애니메이션
- **하단 영역**: 오늘의 추천 학습 (미완료 항목 랜덤 추천)
- **스트리크 배너**: 연속 학습일 + 불꽃 아이콘

#### 4.7.2 하단 네비게이션
- 5개 탭: 홈 / 숫자 / 한글 / 영어 / 게임
- 큰 아이콘(32px) + 라벨, 활성 탭 색상 강조
- 터치 타겟 최소 56px

### 4.8 부모 설정 화면 — P1

- 부모 게이트: 간단한 수학 문제 풀기 (예: "3 + 4 = ?")
- 학습 통계: 카테고리별 진행도, 총 학습 시간, 일별 학습 기록
- 학습 타이머: 일일 학습 시간 제한 설정 (15분/30분/무제한)
- 사운드 설정: 효과음 ON/OFF, TTS 속도 조절
- 데이터 초기화 옵션

### 4.9 스티커북 화면 — P1

- 수집한 스티커 그리드 표시
- 미수집 스티커는 실루엣으로 표시
